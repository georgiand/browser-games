# Snake Game v3.0

A modern, fully-featured Snake game built with HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Difficulty Levels:** Choose between Slow, Medium, Fast, and Insane speeds.
- **Game Modes:** Toggle between "Wrap Mode" (walls are wrap-around) and "Standard Mode" (walls are deadly).
- **High Score Tracking:** Persistent high scores tracked separately for all 8 combinations of difficulty and mode.
- **Day/Night Toggle:** Visual themes for comfortable play.
- **Sound Effects:** Audio feedback for eating and game over states.
- **Pause/Resume:** Ability to pause the game with a visual blur effect.
- **Mobile Support:** Touch/swipe controls optimized for mobile browsers.

## Installation
No installation required. Simply open `index.html` in any modern web browser.

## Controls
- **Desktop:** Arrow keys or WASD to move. Space to pause.
- **Mobile:** Swipe in the desired direction on the game canvas.
